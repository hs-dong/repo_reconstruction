## 创建 fetch_remote_commits.py
- 路径: `eval_script/web_app/fetch_remote_commits.py`
- 功能: 获取 EchoCraft GitHub 仓库的远程 commit 历史
- 支持两种方式:
  1. **GitHub REST API**（需网络访问，支持公开仓库）
  2. **本地 git 仓库**（通过 `origin/main` 远程分支获取，推荐使用）
- 功能特性:
  - 按作者筛选 (`--author`)
  - 按时间范围筛选 (`--since`, `--until`)
  - 输出为 JSON/JSONL 文件 (`--output`)
  - 解析 PR 编号和模块名称
  - 统计信息（按作者、模块分布）
- 测试结果: 成功获取 205 条 commit，与 git_pr_history_report.txt 一致
- 输出文件: `eval_script/result/remote_commits.json`
- 注意: 需兼容 Python 3.6（无 dataclasses, 无 capture_output, 无 fromisoformat）
